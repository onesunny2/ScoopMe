# 🍕ScoopMe(스쿱미)
> 내 주변 식당의 음식을 주문하고, 맛집 정보를 커뮤니티에 공유 및 소통할 수 있는 간편 픽업 서비스

## 앱 소개
- 개발 기간: 25.05.10 ~ 25.06.30 (유지보수 중)
- 구성 인원: iOS 개발(1인), 백엔드(1인), 디자이너(1인)
- 최소 버전: iOS 16.0 +
